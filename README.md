# RickWiki

Вселенная **Rick and Morty** у тебя в кармане.

RickWiki — это минималистичное iOS-приложение для удобного просмотра персонажей, локаций и эпизодов культового сериала. Вся информация подтягивается из официального API и обновляется на лету.

## Что внутри

👥 **Персонажи** — 826 героев с аватарками, статусом, видом и происхождением. От Рика и Морти до самых безумных второстепенных персонажей.

🌍 **Локации** — 126 мест во всех измерениях: планеты, космические станции, измерения и не только.

📺 **Эпизоды** — 51 серия с датами выхода, кодами и списком появившихся персонажей.

## Особенности

- **Infinite scroll** — листай без конца, контент подгружается сам
- **Pull-to-refresh** — обнови одним движением
- **Детальные карточки** — полная информация по каждому персонажу, локации и эпизоду
- Нативная SwiftUI-архитектура — летает на iOS

## Стек

SwiftUI · async/await · MVVM · Rick and Morty API
